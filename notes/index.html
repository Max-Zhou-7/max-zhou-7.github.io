PyTorch Geometric Data Loader Gotcha

January 11, 2025

When using PyTorch Geometric's DataLoader, make sure your Data objects have the correct shape. A common error:

RuntimeError: Expected edge_index to be of shape [2, num_edges]

Solution: Always transpose edge indices before creating Data object:

edge_index = torch.tensor(edges, dtype=torch.long).t().contiguous()

Flask Template Paths in Serverless

January 8, 2025

When deploying Flask to Vercel, template paths can break. Add this to your Flask app init:

import os
template_dir = os.path.join(os.path.dirname(__file__), 'templates')
app = Flask(__name__, template_folder=template_dir)

Debugging Gradient Flow in Neural Networks

December 30, 2024

Quick check for vanishing gradients in PyTorch:

for name, param in model.named_parameters():
    if param.grad is not None:
        print(f"{name}: {param.grad.abs().mean()}")

If gradients are < 1e-7, you likely have vanishing gradient issues.

MongoDB Aggregation for Grouped Queries

December 22, 2024

Useful aggregation pattern for grouping by date:

db.collection.aggregate([
  {
    $group: {
      _id: { $dateToString: { format: "%Y-%m-%d", date: "$createdAt" } },
      count: { $sum: 1 }
    }
  },
  { $sort: { _id: -1 } }
])

Handling SMILES String Encoding

December 18, 2024

When working with molecular SMILES, always validate before processing:

from rdkit import Chem

def is_valid_smiles(smiles):
    mol = Chem.MolFromSmiles(smiles)
    return mol is not None

Invalid SMILES can crash your entire training pipeline.
